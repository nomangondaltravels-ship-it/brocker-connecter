# Broker Connector

A static broker-to-broker real estate website with:

- main broker network site in `index.html`
- separate admin portal in `admin.html`

## Deploy on Vercel

1. Push this folder to a GitHub repository.
2. Import the repository in Vercel.
3. Keep the project as a static site with the root directory set to the repository root.

## Files

- `index.html` - main website
- `admin.html` - admin portal
- `pas/Github.png` - project asset
